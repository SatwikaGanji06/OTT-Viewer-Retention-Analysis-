# OTT-Viewer-Retention-Analysis-
Analyzed 33k episodes across 2.5k shows to diagnose a 48.2% viewer drop-off (8.2pp above benchmark); used Random Forest (89.4% accuracy) and K-means to identify cognitive load &amp; weak hooks as key drivers, flagging 41.4% content as at-risk and proposing $164M-value interventions.
